### Automatic training using deepseek-finetune-train-pod.yml:
0. Set the train_dir and num_epochs in deepseek-finetune-train-pod.yml, e.g.: "./start_finetune.sh og 6"
1. Apply the pod that automatically starts the training:
### AUTOMATIC TRAINING ### kubectl apply -f deepseek-finetune-train-pod.yml

### Manual training via bash
0. OPTIONAL - Apply the pods without automatic setup:
kubectl apply -f deepseek-finetune-pod.yml

1. Setup the pod:
./setup.sh

2. Start bash in pod:
kubectl -n s85468 exec -it deepseek-finetune -- bash

4. Navigate to correct directory:
cd /storage/text2gloss-finetune

4.5. Test:
python test.py

5. Train:
python train.py

5. FINALLY - Remove the pod from the cluster:
exit
kubectl delete -f deepseek-finetune-pod.yml
OR
kubectl delete -f deepseek-finetune-train-pod.yml
