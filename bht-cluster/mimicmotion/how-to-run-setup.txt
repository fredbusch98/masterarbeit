0. (OPTIONAL) Run the pod without setting everything up automatically:
kubectl apply -f mimicmotion-pod.yml

1. Run the shell script to trigger the setup of the pod:
./setup.sh

2. Start bash in mimicmotion pod:
kubectl -n s85468 exec -it mimicmotion -- bash

3. Activate conda environment:
conda activate mimicmotion && cd /storage/MimicMotion

4. Model inference:
python inference.py --inference_config configs/test.yaml
python inference.py --inference_config configs/test-openpose.yaml
python inference.py --inference_config configs/test-openpose-satz.yaml
python inference.py --inference_config configs/test-interpolated.yaml

5. FINALLY - Remove the pod from the cluster:
exit
kubectl delete -f mimicmotion-pod.yml
